# Medicine Price Tracker

## Project Description

This project is a web-based application that allows users to search for medicines and compare their prices and availability across platforms such as PharmEasy, Netmeds, Tata 1mg, and Apollo Pharmacy. It uses web scraping techniques to fetch real-time data and display the best available options.

The backend is built using FastAPI and the frontend using React. The project integrates a complete DevOps pipeline including GitHub, Jenkins for CI/CD, Docker for containerization, Kubernetes for orchestration, Ansible for configuration management, and ELK Stack for monitoring and logging.

The system ensures automated deployment, scalability, and real-time updates, demonstrating a complete Software Development Life Cycle (SDLC) using DevOps practices.

## Uniqueness About the Project

This project addresses a real-world problem where medicine prices vary significantly across platforms like PharmEasy, Tata 1mg, Netmeds, and Apollo Pharmacy, and no unified comparison system exists.

Its uniqueness lies in combining real-time web scraping with a complete DevOps pipeline. It can be extended with advanced features such as a machine learning-based medicine recommendation system, stockout prediction, secure credential management using Vault, infrastructure automation using Ansible, and auto-scaling with Kubernetes HPA.

The project goes beyond typical academic work by integrating scalable and automated deployment using DevOps practices.
